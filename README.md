# uoa-karate

Website for the Auckland University Karate Club, written in Vue.

Currently hosted at [https://www.uoa-karate.club/](https://www.uoa-karate.club/)


## Running the site


#### Install

 `npm i`
 
 #### Serve
 
 `npm serve`
 
 #### Build 
 
 `npm build`
